Calibration with Python
========================

video.py
---------

This is a simple program to grab images from a camera and either save frames from it or  grab video sequences.

Use cases:
- grab images from camera
 -- live
 -- saved mv4v
 -- define image size
- save 
 -- single frames png
 -- save video mg4v
- correct images with calibration matrices
- display video
