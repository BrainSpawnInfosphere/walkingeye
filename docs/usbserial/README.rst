PmodUSBUART Pinout
-------------------------

=== ====== ===========================
Pin Signal Description
=== ====== ===========================
1   RTS    Ready to Send
2   RXD    Receive
3   TXD    Transmit
4   CTS    Clear to Send
5   GND    Ground
6   3V3    Power
=== ====== ===========================
